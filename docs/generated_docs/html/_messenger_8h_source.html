292 /* Add a friend.
293  * Set the data that will be sent along with friend request.
294  * address is the address of the friend (returned by getaddress of the friend you wish to add) it must be FRIEND_ADDRESS_SIZE bytes. TODO: add checksum.
295  * data is the data and length is the length.
296  *
297  * return the friend number if success.
298  * return -1 if message length is too long.
299  * return -2 if no message (message length must be >= 1 byte).
300  * return -3 if user's own key.
301  * return -4 if friend request already sent or already a friend.
302  * return -6 if bad checksum in address.
303  * return -7 if the friend was already there but the nospam was different.
304  * (the nospam for that friend was set to the new one).
305  * return -8 if increasing the friend list size fails.
306  */
